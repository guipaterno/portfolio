Habilidades

HTML

linguagem de marcação usada na criação e estruturação de páginas da web. Ele define a organização do conteúdo, como textos, imagens e links, permitindo sua exibição adequada nos navegadores.

CSS

linguagem usada para estilizar o layout e a apresentação de páginas da web, controlando aspectos visuais como cores, fontes, espaçamento e posicionamento de elementos.

JavaScript

linguagem de programação usada para adicionar interatividade e funcionalidades dinâmicas às páginas da web.

TypeScript

linguagem de programação que adiciona recursos como tipagem estática opcional e suporte a classes e interfaces, facilitando o desenvolvimento de aplicativos JavaScript de grande escala.

React

biblioteca de JavaScript que simplifica o desenvolvimento com componentes reutilizáveis e uma abordagem declarativa para a criação de interfaces.

React Native

framework de desenvolvimento de aplicativos móveis que permite aos desenvolvedores criar aplicativos para iOS e Android usando JavaScript e React.
